Brochure Design

Volunteer project 2018

About project

This project for the actual client, my tutor suggested me to design the trifold brochure for Literacy Aotearoa Wellimgton. I did two presentation to the client with the employment advisor from the Yoobee design school.

Project overview

This project is based on UX design, my client wanted to get a brocher for who has a difficulty to read and write. I didn't use lots of text content for this brochure design, and have done lots of refinements, also used illustration to give a "Welcome" feeling.

Program used

These are illustrations which I designed for the brochure, I wasn't able to use the text contents, so my design concept was the freindly illustration brochure.

I also used orange skin colour palette for no special identity.
