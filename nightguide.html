Night Guide

School project 2018

About project

This project was for the Yoobee design school assignment, the concept for this project was the local council application to get more young people using the local recreational facilities. I've chosen the space place to design the stargazing application, also I uploaded the google survey on the Wellington live facebook commuity.

Project overview

This project is based on UX design, to get a real answers from the local people, and learnt about wireframe, paper prototype and user testing. I relly had fun with the paper prototype and I have still them at home. I also design all UI design as well, icon design which is my another favorite!

Program used

These are the logo design development, the logo design concept wasn't changed from the begining, I want use a morepork as a my app logo, only changed the detail. I really liked the first one as simple, but it was not good enough when I worked on illustrator. So, defined more detail design and not look like fish(2nd&3rd).

The last one is my final logo design, detailed wings and tails which not looks fish anymore.

UI icon design, I really enjoyed this part. I didn't have to design all icons, but I fell in love with icons so design all UI in my applications. The fifth icon on top from the left one is special for
New Zealand, I had to design the Maori ethnicity in my design concept and this icon is represented "Matariki".
