Whats Sup?

EDA group project

About project

What's sup is a location based progressive web application that allows homeless, displaced or anyone in a rut the ability to find services to help in the Wellington region. This app was designed to be used on mobile device while a person is moving between free wifi zones so it keeps certain level off information off line that can be referred back to if need be.

For providers it helps them get in contact with the people who need them in quick and easy way. It also allows providers and services the ability to live update the availablity of goods or services so a person as better idea of wiat tiems or availblity ahead of time. This can give a better indication of wait times and avoids the dissappointment that come from missing out on assistances.

Project overview

This project was the other group's project at EDA. I was called to improve their application, I started the branding concept and a lo-fi wireframe.

Program used

This is the original sup web application. The main problem was the header navigation, it was disappeared when a user scrolle down. UI icon had no consistency. I created all icon for this application after completed lo-fi wireframe. The original map colour palette was too bright and it's not good contast with their original icon.


I focused usability and consistency to improve this web application.

This is lo-fi wireframe, thier side menu was directly popped up when a user visited thier site because they wanted to show thier logo and icon information. I suggested them to make landing page and show the logo and Sup's information before visiting the homepage. The nav bar location is also changed, as this mobile web application and botttom nav bar is much easier to use.


I suggested them to add help icon to find out the tutorial and icon information if the user get confused what this website for and when they are using it.
