EDA Phase-0 Blog

About project

This was EDA's phase-0 project, I was following their curriculum right after Yoobee design school. I learned basic HTML5, CSS3, and Wordpress but this was the first time to build an entire webpage from nothing.

Project overview

I prepared myself with 10 weeks worth of self-learning, coding exercises and developed this GitHub blog. I used Skeleton and Bootstrap in the beginning but I became enjoying CSS Grid and FlexBox. I developed my blog with a manual CSS layout without other built framework.

Software Languages used

My blog layout is similar to my portfolio website because this final version developed right after this website. I didn’t use too much JS function for this blog. The only Header appears when a user is scrolling up.


Blog menu has all the technical stuff what I have learned through 10 weeks and something for cultural stuff about the meditation.


Projects menu has 3 JS projects, Dogsweeper, Calculator and Devil Fruit.
