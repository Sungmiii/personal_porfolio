Wedding Invitation

Personal project 2018

About project

This project for the personal used, I have currently planning my wedding in Korea in 2019. I still have a plenty of time but this is happening in overseas so everything must be ready by March 2019. I started to design an invitation first, my design concept is natural and feel an autumn. I used three type-face : serif & sans serif and handwriting.

Project overview

Stationary design is my favorite, and this is for the people whoes are my family,freinds and relatives, but there is problem that I need to design for 2 languages(English & Korean). These 2 countrys invitation style quite different, so research was very important for completing this project.

Program used

White lily is my favorite flower, I illustrated them on photoshop but noticed that looks "Spring wedding invitation". So I went to the flower shop to find out which flowers good for the autumn, and this was the first time to buy the flowers for myself.

Following illustrations are the autumn style compare to the lily illustration. I personally don't like use too much vivid color, so only used red for the balance.
