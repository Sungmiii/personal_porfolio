RSVP Wedding Website

EDA personal project

About project

I learned basic HTML5, CSS3 and Wordpress at Yoobee design school but I wanted to build my RSVP wedding website from the foundation. EDA was the best place to get all fundamental web development skills. I built my RSVP on React, SQLite3, node.js, express ...etc.

Project overview

This project was for 3weeks personal project, and I really wanted to build something I could use. I used React boiler plate which EDA offered but there was Back-End data connection error with Client side, so I started with npx create-react-app but there was another issue. I built this site after 4 times failure but managed to connect back-end data to clinet API(It was 4 days before to be presented!


I am still working on the Blog menu for the guests who are flying to Korea to attend my wedding. It will be updated on April.

Software Languages used

This website is a very simple layout includes a flower illustration from my wedding invitation, a counting timer on the homepage and the RSVP button. RSVP form includes the ceremony detail and user's important information. All users data is going to the back-end database, and GET request page shows all data to me.


There is an additional function to show 2 different messages depends on the user's Yes/No checkbox. I am using Heroku server, the message user name is updated little bit slowly. Also I used reduce function to get a total guest number.
