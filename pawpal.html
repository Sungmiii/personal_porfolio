Pawpal

School project 2018

About project

PAWPAL is an IoT pet feeder and GPS device that you are able to control from your smart device if your device is connected to WI-FI. PAWPAL application let you check on your pet through the camera attached to the feeder, also you are able to track your petâ€™s location with GPS collar. GPS collar has a sensor in it, the sensor is connected with the feeder so your pet could eat whenever they want but you can customize your feeder settings.

Project overview

This project is based on UX design, I uesd the google survey to get what the potential customers want to use with Pawpal application. After survey, I followed the UX procsee. Sketch, wireframe, paper prototype, Hi-fi prototype and A/B testing.

Program used

These are Pawpal sketches, a user meet the landing page first and they are able to login with google/facebook id. The user need to register their Pawpal device then register their pet details. The instruction pop up page shows up when the user completed all the task. The map is on the homepage as the user want to check where is their pet and this is the main concern from the servey.

Paperprototype! This is my favorite processing in UX design and this method is really useful for the user testing. I gave the crucial tasks to the user and wait until they complete. If the user didn't finish the task easily I fixed the problem and did the other tests. Also, I have done multiple test with different users each time.
